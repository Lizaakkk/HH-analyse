# HH-analyse
Расположение файла с данными в формате CSV: https://drive.google.com/file/d/1Kb78mAWYKcYlellTGhIjPI-bCcKbGuTn/view?usp=drive_link  
Цель проекта:  
- Создать модель, которая бы автоматически определяла примерный уровень заработной платы, подходящей пользователю, исходя из информации, которую он указал о себе.
  
Проект состоит из 4 частей:  
- базовый анализ структуры данных  
- преобразование данных  
- разведывательный анализ  
- очистка данных
